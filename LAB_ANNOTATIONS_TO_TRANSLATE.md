# LAB VALUE ANNOTATIONS TO TRANSLATE

All cases need lab annotations translated from English to Russian

Common annotations:
- (elevated) → (повышено)
- (markedly elevated) → (значительно повышено)
- (mildly elevated) → (умеренно повышено)
- (normal) → (норма)
- (anemia) → (анемия)
- (critical hyperkalemia) → (критическая гиперкалиемия)
- (hypoglycemia) → (гипогликемия)
- (acute renal failure) → (острая почечная недостаточность)
